Portfolio Project Manager in C

Description

Portfolio Project Manager is a based on CRUD (Create, Read, Update, Delete) application developed using C programming. The system helps users manage and organize software development projects by storing project details such as Project ID, Project Name, Technology Use, Project Status, and GitHub Link.

The application provides an easy-to-use menu-driven interface that allows users to add, view, search, update and delete project records efficiently.


Features
- Add New Project
- View All Projects
- Search Project by ID
- Update Project Details
- Delete Project Records
- Project Statistics Dashboard


Technologies Used
- C Programming Language
- Structures
- Functions
- Arrays
- CRUD Operations

Project Objective
The objective of this project is to provide a simple and efficient system for managing software project information while demonstrating  C programming concepts such as structures, functions, arrays, and data management.

Future Enhancements
- File Handling for Permanent Data Storage
- Project Sorting and Filtering
- Project Deadline Tracking
- Report Generation
- User Authentication
